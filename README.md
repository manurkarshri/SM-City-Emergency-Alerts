# CITY EMERGENCY ALERTS V3

## V3 Focus
The app prioritizes:
1. Pinned alerts from alerts.json
2. Official government / agency alert searches
3. IMD and public warning news
4. Police / traffic / road disruption alerts
5. Flood, fire, power, medical, AQI and earthquake-related live news

## Deploy
Upload extracted contents to repository root.

## Notes
No paid APIs are used. The live feed uses public Google News RSS through free CORS/RSS proxy paths and direct official dashboard links.
