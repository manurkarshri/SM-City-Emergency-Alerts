# SM-City-Emergency-Alerts V2

Deployable GitHub Pages PWA for Pune and nearby emergency navigation.

## Includes
- Live emergency feed using free Google News RSS proxies
- Manual alerts via alerts.json
- Region and locality filtering
- Emergency Control Centre tab
- Incident timeline
- Most affected area scoring
- Official source dashboard
- Official X channel links
- Emergency map shortcuts
- Emergency dial page
- Safety checklist
- PWA install support
- Offline cache
- No paid APIs

## Deploy
Upload all extracted files and folders to repository root.
GitHub Settings → Pages → Deploy from branch → main → /(root)

## Manual alerts
Edit alerts.json.
Priority: high = Emergency, medium = Warning, low = Advisory
